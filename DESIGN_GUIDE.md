# 🐝 Redesign Visual - Banco dos Marimbondos

## 🎨 Nova Paleta de Cores - "Marimbondos Theme"

O app foi completamente redesenhado com cores e elementos que remetem a **abelhas** 🐝 e à **natureza** 🌿.

### Cores Primárias

| Característica | Claro | Escuro | Hex |
|---|---|---|---|
| **Ouro (Primária)** | Brilhante | Nas bordas | `#FFB800` |
| **Âmbar Escuro** | Para hovers | Para destaques | `#F59E0B` |
| **Verde Natureza** | Detalhes | Destaques verdes | `#10B981` |
| **Preto (Terciário)** | Texto | Fundos | `#1F2937` |

### Tema Claro - "Dia Ensolarado" ☀️
```
Fundo:          #FFFBF0  (Branco quente, como mel)
Cards:          #FFFFFF  (Branco puro)
Bege:           #FEF3C7  (Tons quentes suaves)
Texto:          #1F2937  (Preto profundo)
Bordas:         #FFB800  (Ouro)
Sombras:        Âmbar com 10% opacidade
```

**Paleta Visual**: Clara, calorosa, como o mel das abelhas ✨

### Tema Escuro - "Noite nas Flores" 🌙
```
Fundo:          #0F0E0C  (Preto morno)
Cards:          #2A251F  (Marrom quente)
Texto:          #FEF3C7  (Bege claro)
Bordas:         #FBBF24  (Ouro claro)
Sombras:        Âmbar com 20% opacidade
```

**Paleta Visual**: Escura mas aconchegante, como uma colmeia aquecida 🐝

### Elementos Temáticos

#### 🐝 Abelhas
- Logo emoji: `🐝` (flutuante com animação `bee-float`)
- Animação de flutuação suave e natural

#### 🌼 Flores & Natureza
- Padrões de fundo com gradientes circulares
- Linhas diagonais sutis (padrão honeycomb)
- Cores verdes como acentos

#### 🟨 Ouro & Mel
- Botões principais em ouro (#FFB800)
- Destaque em âmbar para elementos ativos
- Efeito `glow` nas abas ativas

---

## ✨ Animações Adicionadas

### 1. **Honeycomb** 🟨
- Rotação infinita com escala
- Representa o movimento cíclico da nature

### 2. **Glow Effect** ✨
- Brilho pulsante em âmbar
- Aplicado nas abas ativas
- Remete ao brilho das abelhas

### 3. **Float Animation** 🐝
- Logo flutuando suavemente
- Movimento natural e orgânico
- Aplicado ao emoji da abelha

### 4. **Transições Suaves** 🌊
- Todas as mudanças em 0.4s
- Cubic-bezier para naturalidade
- Sem "saltos" abruptos

## 🌈 Exemplos de Uso

### Botão de Login
```
Claro:  Fundo ouro (#FFB800) → Branco (#FFFFFF) no hover
Escuro: Fundo ouro (#FBBF24) → Âmbar (#F59E0B) no hover
```

### Abas de Navegação
```
Inativa: Texto cinza, sem fundo
Ativa:   Fundo bege (claro) / ouro (escuro) com glow
Hover:   Fundo âmbar com transição suave
```

### Cards de Dados
```
Claro:  Branco puro com borda ouro sutil
Escuro: Marrom quente com gradiente âmbar
```

## 📊 Estatísticas do Design

- **Cores Primárias**: 4 (Ouro, Verde, Preto, Âmbar)
- **Animações**: 8+ (float, glow, honeycomb, etc)
- **Temas Suportados**: 2 (Claro + Escuro)
- **Transições**: 0.4s padrão (suave)
- **Contraste WCAG**: AA+ em ambos os temas
